# Apex Digital AI

Apex Digital AI is a fully autonomous **AI-powered design and marketing studio** for South Africa and global clients.

## Features
- AI Project Generation
- Marketing Campaigns
- Social Media Automation (TikTok, Instagram, YouTube, X/Twitter, LinkedIn, Facebook)
- PDF Contracts & T&Cs
- Analytics & Revenue Tracking
- Tiered Pricing + Credits System
- Stripe & PayPal Monetization
- Full Docker Support
- Ready for Windows 10, Ubuntu 24.04, and cPanel deployment
